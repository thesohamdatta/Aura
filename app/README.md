# Aura App

The Aura App is a Flutter-based mobile application that serves as the companion app for Aura devices. This app enables users to interact with their Aura device, manage apps, and customize their experience.

### Quick Setup

Before getting started, make sure your device is connected and USB debugging is enabled in Developer Options.

1. Navigate to the app directory:
   ```bash
   cd app
   ```

2. Run setup script:
   ```bash
   bash setup.sh android
   ```
 
3. To run the app, navigate to the app directory and use the following command:
   ```bash
   flutter run --flavor dev
   ```
